# Icons

This folder should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- Format: PNG with transparency
- Style: Should represent a form/document/clipboard theme
- Colors: Recommended to use a purple/blue gradient theme (matching the popup design)

## Quick Icon Generation

You can create icons using:

1. Online tools like [Favicon.io](https://favicon.io/) or [Canva](https://www.canva.com/)
2. Design tools like Figma, Adobe Illustrator, or GIMP
3. AI image generators

Suggested icon concept: A clipboard with a checkmark, or a form with a tracking symbol.

## Temporary Solution

If icons are missing, Chrome will use a default extension icon. The extension will still function correctly.
